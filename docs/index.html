Halcyon is the game created by everyone in the Media Arts Game Design course (MEDIAART 3L03, taught by David Ogborn) at McMaster University (Winter 2025). Travel through time, face ice, fire, and water, and complete challenges to create a better future! Created with Godot game engine and Blender.
